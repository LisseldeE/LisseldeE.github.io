# TokenPeek - AI额度实时监控工具

## 项目简介

TokenPeek 是一个轻量级系统托盘工具，帮助 AI 模型使用者实时掌握 API 额度消耗情况，避免因额度不足中断工作流。常驻系统托盘，需要时单击展开，点击空白处自动隐藏。聚合视图查看所有已配置模型的余额、用量、调用次数。所有配置和缓存数据存储在本地，无云端上传。

## 功能特性

### 核心功能
- **实时监控**: 实时显示 API 余额、今日已用
- **托盘常驻**: 常驻系统托盘，单击展开面板，点击空白处自动隐藏
- **自动刷新**: 定时自动刷新余额数据，支持手动刷新
- **状态预警**: 根据余额状态自动切换图标颜色（绿色充足、黄色注意、红色告急）
- **首次启动向导**: 首次启动引导配置 API Key

### 界面交互
- **简洁面板**: 一个面板查看所有核心指标
- **快速刷新**: 展开时自动刷新一次，右上角提供手动刷新按钮
- **刷新动画**: 平滑的细长进度条动画，提供视觉反馈
- **简约图标**: 使用简约线条符号图标，专业现代

### 配置管理
- **API Key 管理**: 支持修改和验证 API Key
- **刷新间隔**: 支持 1/5/15/30 分钟刷新间隔设置
- **预警阈值**: 支持自定义余额预警阈值
- **配置导入导出**: 支持导出/导入配置文件，方便多设备同步

### 多语言支持
- **中英文切换**: 支持中文/英文界面实时切换
- **自动语言识别**: 自动识别系统语言，默认显示对应语言

## 使用方法

### 首次启动

1. 双击运行 `TokenPeek.exe` 或 `python TokenPeek.py`
2. 弹出配置向导窗口
3. 选择模型平台
4. 输入 API Key
5. 点击「验证并保存」
6. 验证通过后自动进入主界面

### 日常使用

1. 单击托盘图标展开面板
2. 查看余额、今日已用
3. 点击空白处自动隐藏面板
4. 右键托盘图标可快速刷新、打开设置、关于等

### 配置修改

1. 点击面板设置按钮
2. 修改 API Key、刷新间隔、预警阈值
3. 切换界面语言（中文/英文）
4. 导出/导入配置文件
5. 关闭配置窗口自动保存

## 技术特性

### 数据安全
- **本地优先**: 所有配置和缓存数据存储在本地（~/.TokenPeek/）
- **无云端上传**: 不上传任何数据到云端服务器
- **隐私保护**: API Key 明文存储，本地加密可选

### 稳定性保障
- **优雅降级**: API 不可用时不影响程序运行，显示旧数据并提示
- **错误处理**: API 调用失败时显示黄色/灰色图标，不崩溃
- **超时机制**: 5 秒超时，超时后显示旧数据 + 状态提示

### 性能优化
- **内存占用**: < 50MB（常驻）
- **启动速度**: 点击托盘图标到面板显示 < 200ms
- **界面响应**: 平滑动画，无卡顿
