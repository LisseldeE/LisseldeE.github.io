使用指南

🏠 创建房间（主机端）
1 点击"创建连接"按钮
2 可选：设置6位数字房间号和密码保护
3 点击创建，进入同步状态界面
4 等待其他用户加入，开始实时同步
主机端可查看所有同步记录、在线连接端列表
🔗 加入房间（连接端）
1 点击"加入连接"按钮
2 扫描已有房间或手动输入房间号
3 如有密码保护，输入密码进行预验证
4 验证失败即时提示，可修改信息重试
5 点击连接，自动从主机全量同步
断线重连后自动增量同步补齐差异

核心功能

实时同步
文件添加、修改、删除、重命名操作实时同步至所有连接端，支持文件夹递归同步
传输可靠性
断点续传、完整性校验、背压自适应、流式传输，确保大文件稳定传输
多端同步
主机端修改实时同步至所有端，支持并发传输限制（最多3个文件）
安全验证
自定义房间号，可选密码验证，加入时版本号验证确保连接兼容性
文件操作
支持复制、剪切、粘贴、删除、重命名，双击空白快速添加文件/文件夹
文件预览
双击文件直接预览，预览窗口只读模式，避免误修改文件内容
多语言支持
支持中文/英文界面切换，实时切换无需重启程序

技术栈

Python 3.x PySide6 (Qt6) TCP 自定义协议 流式传输 断点续传

核心功能

实时监控
实时显示 API 余额、今日已用，聚合视图查看所有已配置模型的余额、用量
托盘常驻
常驻系统托盘，单击展开面板，点击空白处自动隐藏，不占用主界面空间
自动刷新
定时自动刷新余额数据，支持手动刷新，展开时自动刷新一次
状态预警
根据余额状态自动切换图标颜色（绿色充足、黄色注意、红色告急）
欢迎向导
首次启动引导配置模型平台和 API Key，轻松完成初始化设置
模型配置
支持配置多个模型平台（DeepSeek、Kimi、StepFun），自定义模型信息
多语言支持
支持中文/英文界面实时切换，自动识别系统语言
数据安全
所有配置和缓存数据存储在本地（~/.TokenPeek/），无云端上传
