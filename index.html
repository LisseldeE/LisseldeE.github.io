LANSyncBox

局域网文件实时同步工具，实现多人隔空文件共享

GitHub 仓库
Microsoft Store
其他下载
GitHub 发行 Gitee 发行

核心功能

🔄 实时同步
文件添加、修改、删除、重命名操作实时同步至所有连接端，支持文件夹递归同步
🛡️ 传输可靠性
断点续传、完整性校验、背压自适应、流式传输，确保大文件稳定传输
👥 多端同步
主机端修改实时同步至所有端，支持并发传输限制（最多3个文件）
🔐 安全验证
自定义房间号，可选密码验证，加入时版本号验证确保连接兼容性
📁 文件操作
支持复制、剪切、粘贴、删除、重命名，双击空白快速添加文件/文件夹
👁️ 文件预览
双击文件直接预览，预览窗口只读模式，避免误修改文件内容
🌐 多语言支持
支持中文/英文界面切换，实时切换无需重启程序

技术栈

Python 3.x PySide6 (Qt6) TCP 自定义协议 流式传输 断点续传

SyncGUI

GitHub 仓库
Microsoft Store
其他下载
GitHub 发行 Gitee 发行

核心功能

🔄 双向同步
支持双端之间的双向文件同步，智能识别新增、修改、冲突文件
🎯 差异检测
MD5 哈希内容比对，自动识别文件差异，精确判断同步方向
📂 目录同步
可配置特定目录按整体同步，正确处理空目录及符号链接
⚡ 单向同步
差异同步或覆盖同步模式，可选保留或删除多余项目
📝 忽略规则
支持自定义忽略规则语法，灵活排除不需要同步的文件
✏️ 自定义名称
支持自定义双端目录名称显示，配置自动保存恢复
🌐 多语言支持
支持中文/英文界面切换，实时切换无需重启程序

技术栈

Python 3.x PySide6 Windows API MD5 哈希
